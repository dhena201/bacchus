I) HTML Structure - top

Background slider

You can use 1 static image or slideshow for background slider. This template uses jQuery Cycle for background slider. Here is the general structure

			<div id="cycle-slideshow">
				<div class="slideimg img1"></div>
				<div class="slideimg img2"></div>
				<div class="slideimg img3"></div>
			</div>

In case you want to use background slider, you need to include jquery.cycle.lite.js plugin inside the page.

<script type="text/javascript" src="js/jquery.cycle.lite.js"></script>

Use the following script in script.js to fire the plugin

$('#cycle-slideshow').cycle();

For more information about this jQuery plugin, please visit to jQuery Cycle

Notify form

BuzzComi allow you 3 type to save user email: Save to database, save to file & intergrate with Mailchimp service.

This is HTML code to use for saving to database or file:

			<form action="processNotifyForm.php" method="post" id="md-subscribe-form" name="mc-embedded-subscribe-form" class="validate" target="_blank" novalidate>
					<div class="insert omega">
							<input type="email" value="" name="email" class="email" id="mce-EMAIL" placeholder="Insert your email here" required>
					</div>
					<div class="">
							<input type="submit" value="Subscribe" name="subscribe" id="md-subscribe" class="md-button notify-submit">
					</div>
			</form>

To config where to save, you need to change processNotifyForm.php file. All you need is change $result = writeToFile($email); to $result = insertEmailToDatabase($email); (remember to change database username, passwords & database name). There's an sql file that help you auto create database table too.

If you like to use Mailchimp services, you can replace above HTML code with Mailchimp code, need modify a little like this:

				<form action="MAICHIMPLINK" method="post" id="mc-embedded-subscribe-form" name="mc-embedded-subscribe-form" class="validate" target="_blank" novalidate>
					<div class="mc-field-group">
						<label for="mce-EMAIL">Email Address </label>
						<input type="email" value="" name="EMAIL" class="required email" id="mce-EMAIL">
					</div>
						<div class=""><input type="submit" value="Subscribe" name="subscribe" id="mc-embedded-subscribe" class="button"></div>
					</form>

II) CSS Files and Structure - top

This theme uses style.css file as main CSS file.

style.css

Main CSS files in this theme is style.css that contains all of specific stylings for the theme.The file is separated into 13 sections using labels as follows:

		
1.	Reset
2.	Common style
3.	Main style
4.	Diaglog style
5.	Footer
6.	Button

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

responsive.css

responsive.css contains styling for media queries that allows us to tailor to different resolutions and devices without changing the content. The file is separated into sections as follows:

{SOME CODES}

}

@media only screen and (max-width: 700px) {

{SOME CODES}

}

@media only screen and (max-width: 480px) {

{SOME CODES}

}

If you would like to edit styling of the specific device width, simply find the appropriate label in the responsive.css file, and then scroll down until you find the appropriate style that needs to be edited.

III) JavaScript - top

This theme imports following Javascript files.

  1. jquery.cycle.lite.js - jQuery cycle
  2. jquery.lwtCountdown-1.0.js - jQuery Count down
  3. jquery.form.js - allows you to upgrade HTML forms to use AJAX
  4. jquery.validate.min.js - validate contact form and comment form
  5. misc.js - contains some helped functions
  6. script.js - contains all custom script

Except script.js, all of the plugins are packed, so you don't need to manually edit anything in these files. The only necessary thing to know is how to call the method. You can refer to the website of the plugin author I listed in the "Sources and Credits" section to learn more about them.
